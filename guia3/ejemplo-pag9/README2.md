#Portal de Recursos Educativos
Este proyecto consiste en una Galería de Recursos Digitales, una página web diseñada para facilitar el acceso a herramientas útiles como bibliotecas virtuales, simuladores y reglamentos internos dentro de una institución educativa.

- **Integrantes:**
- Líder: Elvis.
- Colaboradoras: Yokaira y Rosa.

##**Tecnologías Utilizadas**

- HTML5: Para la definición del contenido y estructura base.
- CSS3: Utilizando Flexbox y CSS Grid para un diseño moderno y responsivo.
- Git & GitHub: Para el control de versiones y el trabajo colaborativo grupal.

##**Dinámica del Proyecto**

- Estructura Base: El archivo index.html define una sección de tarjetas para practicar la adición de nuevos elementos.
- Estilos: El archivo style.css maneja la identidad visual, incluyendo efectos de transición en las tarjetas.
